# PesoPilot Development Roadmap

## Phase 1: Production Hardening

- Add unit and integration tests for auth, ownership checks, reports, and validation.
- Add refresh-token persistence and revocation.
- Add rate limiting and request logging correlation IDs.
- Add CI for typecheck, build, lint, and Prisma validation.

## Phase 2: Mobile UX Completion

- Add edit/detail/delete flows for expenses, income, and budgets.
- Add receipt image picker and upload flow.
- Add skeleton components for each loading surface.
- Add persisted settings and profile updates against the API.

## Phase 3: Analytics

- Add monthly trend aggregation.
- Replace simple chart cards with Victory Native XL visualizations.
- Add exportable reports.
- Add budget alerts and recurring transaction reminders.

## Phase 4: AI Preparation

- Replace placeholder category matching with an explicit AI provider interface.
- Add user-confirmed category suggestions.
- Add privacy controls and audit logging for AI-assisted categorization.
